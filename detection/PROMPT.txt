## THÔNG TIN DATASET

Dataset Structure:

```text
dataset_drink_brand_logo/
└── detection/
    ├── annotations/
    │   ├── img_00001.xml
    │   ├── img_00002.xml
    │   └── ...
    ├── images/
    │   ├── img_00001.jpg
    │   ├── img_00002.jpg
    │   └── ...
    ├── splits/
    │   ├── train.txt
    │   ├── val.txt
    │   └── test.txt
    ├── meta.txt
    └── object_distribution.png
```

Annotation Format:

* Pascal VOC XML
* Một ảnh có thể chứa nhiều object
* Bounding box format:

```xml
<bndbox>
    <xmin>151</xmin>
    <xmax>412</xmax>
    <ymin>203</ymin>
    <ymax>360</ymax>
</bndbox>
```

Object Label:

```text
logo
```

Dataset Statistics:

* Total Images: 2980
* Total XML Files: 2980
* Total Objects: 5995
* Average Objects/Image: 2.01

Image Statistics:

* Image Width Mean: 640.0
* Image Height Mean: 640.0
* Brightness Mean: 120.19

RGB Statistics:

```python
Mean Red Channel   : 132.94
Mean Green Channel : 114.45
Mean Blue Channel  : 116.27
```

Dataset Characteristics:

* Detection dataset for beverage logo detection
* Pascal VOC XML annotations
* Multiple logos per image
* Single detection class in XML: `logo`
* Metadata contains brand-level information:

  * CocaCola
  * Pepsi

Dataset Split:

* train.txt
* val.txt
* test.txt

Mapping Rule:

```python
image_path = images/<name>.jpg
xml_path   = annotations/<name>.xml
```
